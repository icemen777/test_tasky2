# test_tasky2
Исходный код задачника для тестого задания

Представляет из себя список задач, с вариантами сортировки по выполнению, дате и имейлу. 
Есть простейшая система авторизации, админ может из менять статус задач, редактировать и добавлять новые.
Любой пользователь может только добавлять новые задачи.
